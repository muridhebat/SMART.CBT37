Memuat Aplikasi

⚠️
Gagal Memuat Aplikasi
Terjadi error pada script. Kemungkinan ada kode JavaScript yang rusak di file index.html (Google Apps Script).
LANGKAH 1: Buka Link Script Langsung
Buka link Apps Script di browser (bukan lewat GitHub Pages).
URL:
LANGKAH 2: Buka Console (F12)
Tekan F12 → tab Console → lihat pesan error merah.
Jika tidak ada error di Console, masuk ke tab Network → klik file index.html → lihat responsenya.
LANGKAH 3: Perbaiki Error di Kode.gs
Setelah menemukan pesan error, perbaiki di file Kode.gs di Google Apps Script Editor, lalu deploy ulang.
